O que o usuário percebe

O que a máquina percebe

Entrada

O teclado envia um evento. Isso é entrada de dados, não armazenamento definitivo.

Processamento

CPU e RAM trabalham juntas para interpretar, manter e manipular o dado enquanto o programa está em uso.

Armazenamento

Somente ao salvar o dado é escrito em SSD/HD, tornando-se persistente após desligar o computador.
